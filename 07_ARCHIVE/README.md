07_ARCHIVE

Purpose

This folder stores older versions, retired workflows, deprecated prompts, and completed projects.

Nothing important should be permanently deleted. Instead, move outdated materials here for future reference.

Contents

- Old Workflows
- Old Prompts
- Retired Master Studios
- Completed Projects
- Historical Versions

Rules

- Archive instead of deleting.
- Keep a record of why an item was archived.
- Preserve version history whenever possible.
- Use the archive as a reference for future improvements.
