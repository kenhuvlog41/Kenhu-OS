02_MASTER_PROMPTS

Purpose

This folder stores all official KENHU master prompts.

These prompts are the approved versions used repeatedly for AI image generation, AI video generation, product promotion, and content creation.

Contents

- Image Prompts
- Video Prompts
- Thumbnail Prompts
- Studio Prompts
- Caption Prompts

Rules

- Keep only approved prompts.
- Update prompts when a better version is discovered.
- Never delete an old prompt without recording the reason.
- Version important prompts instead of replacing them.
