01_MASTER_STUDIOS

Purpose

This folder stores all official KENHU master studios.

A master studio is a permanent environment used to create consistent AI-generated content.

Contents

- Shoe Studio
- Mukbang Studio
- Fashion Studio
- Product Studio
- Future master studios

Rules

- Do not overwrite an approved master studio.
- Save the final prompt together with the studio.
- Record every improvement as a new version.
- Keep visual consistency across all content.
